# Citing

If you use SmartSepsis-Oph software or design outputs in your work, please
cite as follows.

## Software

Citation metadata is maintained in
[`CITATION.cff`](https://github.com/JVLegend/smartsepsis/blob/main/CITATION.cff).

```bibtex
@software{smartsepsis_oph_software_2026,
  title    = {{SmartSepsis-Oph}: AI-first molecular diagnostics design for ophthalmic infections},
  author   = {Pacheco Dias, Jo{\~a}o Victor and Sakuno, Gustavo},
  year     = {2026},
  version  = {0.1.0-phase0},
  url      = {https://github.com/JVLegend/smartsepsis},
  license  = {MIT}
}
```

## Dataset

The companion multimodal AMR dataset on Hugging Face:

```bibtex
@dataset{smartsepsis_oph_2026,
  title  = {{SmartSepsis-Oph}: Multimodal AMR variants for ophthalmology research},
  author = {Dias, Jo{\~a}o Victor and Sakuno, Gustavo and Primo, Raul},
  year   = {2026},
  url    = {https://huggingface.co/datasets/JVLegend/smartsepsis-oph},
  doi    = {tbd},
  note   = {Dataset accompanying the SmartSepsis-Oph research line, HC-FMUSP × Mass Eye and Ear (Harvard).}
}
```
