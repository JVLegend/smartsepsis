# Contributing

The canonical contributing guide lives in the repository root:
[`CONTRIBUTING.md`](https://github.com/JVLegend/smartsepsis/blob/main/CONTRIBUTING.md).

This page summarizes the highlights.

## What we are looking for

1. **Wet-lab partners** (highest priority) — Cas12a / Cas13 collateral
   cleavage, isothermal amplification (RPA / LAMP), bacterial culture.
2. **Senior scientific advisors** in CRISPR-Dx, ocular microbiology,
   molecular pathology, isothermal amplification chemistry.
3. **Clinical partners** with case volume in endophthalmitis, microbial
   keratitis, or neonatal ophthalmia.
4. **Funders** interested in AI-first diagnostics for neglected sample types.

See [Partners](project/partners.md) for the full version.

## Code contributions

We welcome:

- Bug fixes and reproducibility improvements.
- Documentation and tutorial contributions.
- Independent validation of design pipeline outputs.
- Additions to the guide-RNA design and scoring modules.

**Please open an issue before submitting a substantial PR** so we can align
on scope.

## What we ask of contributors

- **Honesty about scope.** Do not introduce code or documentation that
  overstates the program's stage or makes performance claims not yet
  experimentally validated. See [Disclaimers](disclaimers.md).
- **Reproducibility.** Provide enough information for someone else to run
  what you ran. See [Reproducibility](data/reproducibility.md).
- **Attribution.** Cite methods you borrow from published work.

## Code of conduct

Be excellent to each other. A formal Code of Conduct will be published ahead
of the framework preprint. Until then, the spirit of the Contributor
Covenant applies.
